
# Gerador de Listas de Exercícios com IA

Projeto para a disciplina **Desenvolvimento de Aplicações Web**.

Autores: Gabriel e Tiago

Descrição
---------
Ferramenta web para apoiar professores da educação básica na geração de listas de exercícios personalizadas com auxílio de Inteligência Artificial. A IA sugere enunciados, alternativas, gabarito e explicações; o professor revisa, edita e salva as listas.

Stack (resumo)
---------------
- Frontend: React + Vite (Tailwind CSS)
- Backend (planejado): Python + FastAPI + SQLModel
- Autenticação: JWT (planejado)

Estrutura atual do repositório
------------------------------
- `Trabalho Final Web.md` — descrição do trabalho e cronograma
- `sprint2/frontend/` — código do frontend em React (dados mockados nesta fase)
- `sprint1/` — material da Sprint 1 (plano, protótipos, telas)
- `sprint2/` — artefatos da Sprint 2 (documentação e progresso do frontend)

Como rodar (frontend - mockado)
-------------------------------
No momento o frontend usa dados mockados. Para executar localmente:

```bash
cd frontend
npm install
npm run dev
```

Cronograma
----------
O desenvolvimento segue o cronograma do professor, dividido em 4 sprints:

- Sprint 1: planejamento e protótipo visual
- Sprint 2: implementação do frontend (dados mockados)
- Sprint 3: backend, banco de dados e integração com IA
- Sprint 4: testes, hospedagem e documentação final
